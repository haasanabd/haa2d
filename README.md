# Haa 3D Management v1.0

تطبيق Flutter احترافي لإدارة مشروع طباعة ثلاثية الأبعاد، يعمل بالكامل بدون اتصال بالإنترنت مع قاعدة بيانات SQLite محلية. تم تصميم التطبيق ليكون قابلاً للتطوير والإضافة مستقبلاً، مع التركيز على الكود النظيف والفصل بين الطبقات.

## الميزات الرئيسية

*   **لوحة التحكم (Dashboard)**: عرض شامل لرصيد القاصة، عدد المنتجات، المواد الخام، الزبائن، الموردين، إجمالي المبيعات والمشتريات، والأرباح.
*   **إدارة الزبائن والموردين**: إضافة، تعديل، وعرض تفاصيل الزبائن والموردين مع كشوف حساباتهم.
*   **إدارة المخزون**: تتبع المواد الخام والمنتجات الجاهزة، مع تحديث تلقائي للكميات بعد عمليات البيع والشراء والإنتاج.
*   **فواتير المبيعات والمشتريات**: نظام فواتير كامل يدعم البيع والشراء النقدي والآجل، مع تحديث أرصدة الزبائن والموردين والقاصة والمخزون تلقائيًا.
*   **إدارة الإنتاج**: تسجيل عمليات الإنتاج، خصم المواد الخام، وإضافة المنتجات الجاهزة، مع حساب تكلفة الإنتاج ونسبة الهدر.
*   **القاصة (الصادر والوارد)**: سجل مفصل لجميع الحركات المالية (وارد، صادر، مصروفات)، مع إمكانية التنقل بين التواريخ.
*   **سندات القبض والصرف**: تسجيل سندات القبض من الزبائن وسندات الصرف للموردين، مع تحديث أرصدة الجهات والقاصة.
*   **المصروفات**: إضافة وتصنيف المصروفات المختلفة (كهرباء، إيجار، رواتب، إلخ) وتأثيرها على القاصة.
*   **التقارير**: تقارير احترافية للمبيعات، المشتريات، الأرباح، المخزون، المواد الخام، الإنتاج، المصروفات، وكشوف حسابات الجهات.
*   **البحث والفلاتر**: بحث سريع داخل جميع الصفحات وفلاتر متقدمة حسب التاريخ، الجهة، المنتج، المادة، ونوع العملية.
*   **النسخ الاحتياطي والاستعادة**: نظام كامل للنسخ الاحتياطي والاستعادة لقاعدة البيانات المحلية.
*   **دعم اللغة العربية الكامل (RTL)**: واجهة مستخدم مصممة بالكامل لدعم اللغة العربية.
*   **العمل بدون إنترنت**: التطبيق يعمل بشكل كامل دون الحاجة إلى اتصال بالإنترنت.

## المتطلبات التقنية

*   Flutter SDK (الإصدار المستقر الأخير)
*   Dart SDK
*   Android Studio أو VS Code مع إضافات Flutter

## هيكل المشروع (Clean Architecture)

تم بناء المشروع باستخدام هيكل Clean Architecture لضمان فصل الاهتمامات وقابلية الصيانة والتطوير:

*   **`lib/core`**: يحتوي على الملفات الأساسية مثل الثيمات، الثوابت، الأدوات المساعدة، وإدارة الأخطاء.
*   **`lib/data`**: يحتوي على مصادر البيانات (Data Sources) مثل `DatabaseHelper`، والنماذج (Models) التي تمثل جداول قاعدة البيانات، والمستودعات (Repositories) التي تتعامل مع البيانات.
*   **`lib/domain`**: يحتوي على الكيانات (Entities) التي تمثل الكائنات الأساسية في التطبيق، والواجهات (Interfaces) للمستودعات (Repositories)، وحالات الاستخدام (Use Cases) التي تحتوي على منطق العمل.
*   **`lib/presentation`**: يحتوي على طبقة الواجهة الرسومية (UI) بما في ذلك صفحات التطبيق (Pages)، الودجات (Widgets)، وإدارة الحالة (State Management) باستخدام Bloc/Provider.

## كيفية إعداد وتشغيل المشروع

1.  **استنساخ المستودع (Clone the repository)**:
    ```bash
    git clone <URL_المستودع>
    cd haa_3d_management
    ```

2.  **الحصول على التبعيات (Get dependencies)**:
    ```bash
    flutter pub get
    ```

3.  **تشغيل التطبيق (Run the application)**:
    ```bash
    flutter run
    ```
    أو افتح المشروع في Android Studio/VS Code وقم بتشغيله على محاكي أو جهاز حقيقي.

4.  **بناء ملف APK (Build APK)**:
    ```bash
    flutter build apk --release
    ```
    سيتم إنشاء ملف APK في `build/app/outputs/flutter-apk/app-release.apk`.

## ملاحظات هامة

*   تم تصميم الواجهة لدعم اللغة العربية بشكل كامل (RTL).
*   يجب التأكد من أن جميع الكميات والأسعار المدخلة قيم موجبة.
